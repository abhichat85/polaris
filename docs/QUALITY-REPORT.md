## What this measures

These scenarios verify the agent's *decision making* with a deterministic LLM stub.
They prove the loop, executor, sink, and tools wire together correctly under realistic
agent behavior patterns. They do NOT measure:

- Real LLM output quality (run `scripts/eval-live.ts` with ANTHROPIC_API_KEY for that)
- Live E2B sandbox boot (needs E2B_API_KEY)
- Browser-side preview rendering (needs running dev server)
- End-to-end Stripe webhook flow (needs Stripe test events)

See `docs/MANUAL-TEST-PLAN.md` for the manual checklist + `scripts/eval-live.ts`
for the live-API runner.
